# 安全生产责任制

**Summary**

安全生产责任制是指承包单位必须建立的、覆盖所有岗位的全员安全生产责任体系。该制度要求以正式文件下发，内容与岗位职责匹配，并需交付发包单位备案。此外，承包单位须逐级逐岗位签订安全生产责任书，并接受发包单位的考核督导，确保考核结果与绩效奖惩挂钩。违反相关要求将面临扣分处罚。

## 定义

安全生产责任制是承包单位在项目现场安全管理中的核心制度。它要求：
- 建立覆盖项目所有工作岗位的全员安全生产责任制，并以正式文件形式下发。
- 责任制的内容必须与各岗位的实际工作职责相匹配。

## 适用范围

该制度主要适用于承包单位在发包单位项目中的安全管理活动，包括责任制的建立、备案、责任书的签订、考核及奖惩环节。

## 关键规则

根据相关方安全管理规范，关键规则包括：

- **建立与备案**：承包单位必须建立全员安全生产责任制，并将正式文件交发包单位备案。未能完成备案的，**扣5分**。
- **责任书签订与一致性**：须逐级逐岗位签订安全生产责任书，责任书内容应与岗位安全生产责任制内容匹配。发现不匹配的，**扣2分**。
- **考核制度与执行**：承包单位必须制定全员安全生产责任制的考核制度，并按制度开展考核。**未制定考核制度或未按制度开展考核的，此项不得分**。
- **考核结果应用**：考核结果必须与绩效、奖惩等挂钩。**未实现挂钩的，扣5分**。
- **发包单位职责**：发包单位需督促承包单位开展责任制考核，并将考核结果与绩效奖惩挂钩。

## 检查扣分标准汇总

| 检查项 | 扣分标准 |
| :--- | :--- |
| 未将全员安全生产责任制文件交发包单位备案 | 扣5分 |
| 安全生产责任书与岗位安全生产责任制内容不匹配 | 扣2分（每条） |
| 未制定全员安全生产责任制的考核制度或未按制度开展考核 | 不得分 |
| 考核结果未与绩效、奖惩等挂钩 | 扣5分 |
