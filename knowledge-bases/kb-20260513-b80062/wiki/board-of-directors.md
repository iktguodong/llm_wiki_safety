# 董事会

**Summary**

董事会是公司的执行机构，负责执行股东会决议、决定公司的经营计划和投资方案等重大事项。根据《中华人民共和国公司法》，有限责任公司和股份有限公司均需设置董事会（但规模较小或股东人数较少的公司可仅设一名董事）。董事会由全体董事组成，设董事长一人，可以设副董事长。

**定义**

董事会是公司的常设执行机构，负责执行股东会决议，制定公司经营计划和投资方案，决定内部管理机构设置，聘任或解聘高级管理人员，并制定公司的基本管理制度。董事会代表公司对外进行业务活动，对内管理公司运营。

**适用范围**

- **有限责任公司**：必须设置董事会（或者仅设一名董事），成员为三人以上。
- **股份有限公司**：必须设置董事会（或者仅设一名董事），成员为三人以上。
- **特殊情形**：职工人数三百人以上的有限责任公司，其董事会成员中应当有公司职工代表。

**关键规则**

| 规则类型 | 有限责任公司的规则 | 股份有限公司的规则 |
| :--- | :--- | :--- |
| **组成** | 成员三人以上（职工人数三百人以上的应有职工代表） | 成员三人以上（规模较小可仅设一名董事） |
| **董事长** | 设董事长一人，可以设副董事长；产生办法由公司章程规定 | 设董事长一人，可以设副董事长；董事长由董事会以全体董事过半数选举产生 |
| **董事任期** | 每届不得超过三年，连选可以连任 | 每届不得超过三年，连选可以连任 |
| **会议召开条件** | 全体董事过半数出席方可举行 | 全体董事过半数出席方可举行 |
| **决议通过条件** | 全体董事过半数通过 | 全体董事过半数通过 |
| **表决机制** | 一人一票 | 一人一票 |
| **会议频率** | (现有资料未明确最低次数) | 每年度至少召开两次会议 |
| **临时会议提议** | (现有资料未明确) | 代表十分之一以上表决权的股东、三分之一以上董事或监事会可提议召开临时会议 |
| **董事辞任** | 书面通知公司，通知送达当日生效 | (暂无具体条文) |
| **董事解任** | 股东会可以决议解任董事，决议作出之日解任生效；无正当理由提前解任的，董事可要求赔偿 | (暂无具体条文) |
| **董事责任** | (暂无具体条文) | 董事应对董事会决议承担责任；如决议违反法律、法规或章程并造成严重损失，参与决议的董事需负赔偿责任。但经证明在表决时曾表示异议并记载于会议记录的，该董事可以免除责任 |

**董事会的主要职权**

根据《中华人民共和国公司法》第六十七条，董事会行使以下职权：
1.  召集股东会会议，并向股东会报告工作；
2.  执行股东会的决议；
3.  决定公司的经营计划和投资方案；
4.  制订公司的利润分配方案和弥补亏损方案；
5.  制订公司增加或者减少注册资本以及发行公司债券的方案；
6.  制订公司合并、分立、解散或者变更公司形式的方案；
7.  决定公司内部管理机构的设置；
8.  决定聘任或者解聘公司经理及其报酬事项，并根据经理的提名决定聘任或者解聘公司副经理、财务负责人及其报酬事项；
9.  制定公司的基本管理制度；
10. 公司章程规定或者股东会授予的其他职权。
