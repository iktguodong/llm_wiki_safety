# 管理机构和人员

**Summary**  
该页面定义了对承包单位安全管理机构设置、安全管理人员能力及资质类人员配备与持证的要求，以及对应的检查标准和扣分规则。

## 定义  
管理机构和人员是指承包单位按项目设立的安全管理机构或专职安全管理人员（含现场），以及安全管理人员的安全生产知识和管理能力考核合格证明，还包括危险货物装卸管理人员、特种设备作业人员、特种作业人员、驾驶员、消防设施操作员等资质类人员。

## 适用范围  
适用于发包方对承包单位安全管理机构及资质类人员配置、持证上岗情况的检查与监督。

## 关键规则  

### 机构与人员设置
- 承包单位须按项目设立安全管理机构或专职安全管理人员（含现场），数量应与项目规模相适应。
- 安全管理人员须持有有效安全生产知识和管理能力考核合格证明，并报发包方备案。

### 资质类人员要求
- 资质类人员包括：危险货物装卸管理人员、特种设备作业人员、特种作业人员、驾驶员、消防设施操作员等。
- 上述人员须持证上岗，并在资质范围内从事作业活动。
- 发包方应根据业务发包的具体情况，明确承包单位须提供的资质类人员的类别及数量，并对持证上岗情况进行监督检查。

### 扣分标准（检查）
| 情形 | 扣分 |
|------|------|
| 未按项目设立安全管理机构或专职安全管理人员 | 本项不得分 |
| 安全管理人员数量不符合项目规模要求 | 扣3分 |
| 安全管理人员无有效安全生产知识和管理能力考核合格证明 | 本项不得分 |
| 安全管理人员能力经考核不合格 | 扣2分 |
| 资质类人员类型或数量不符合要求 | 扣3分 |
| 相关作业人员未持证上岗 | 扣2分 |
| 相关作业人员从事作业活动不在资质范围内 | 扣5分 |
