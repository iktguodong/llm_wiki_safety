## 四、部署建议

### 立即可部署

✅ **生产就绪** (Alpha)

```bash
# 后端启动
cd backend
python -m uvicorn app:app --reload --host 0.0.0.0 --port 8000

# 前端启动
cd frontend
npm run dev

# 访问
http://localhost:5173
```

### 前置条件

1. **配置 API Key**
   - 在设置页面配置 DeepSeek/OpenAI API Key
   - 点击"测试"验证连接

2. **初始化知识库**
   - 创建知识库
   - 上传文档 (PDF/Word/Markdown)
   - 等待自动解析
